# ДОСЬЕ: Stripe (Bridge)

**Статус:** Новый игрок с мощным backing  
**Год запуска:** 2026  
**Штаб-квартира:** США  
**Тип:** USDC интеграция для Stripe экосистемы

## ОСНОВНАЯ ИНФОРМАЦИЯ

### Позиционирование
- USDC платежи для существующих клиентов Stripe
- Унифицированная панель для фиат и крипто платежей
- Минимальные изменения в существующих workflow

### Поддерживаемые активы
- **Основной фокус:** USDC стейблкоин
- **Сети:** Ethereum, Solana, Polygon, Base
- **Дополнительно:** USDP, USDG (расширение стейблкоинов)
- **Стратегия:** Качество над количеством, фокус на стабильности

## ПРОДУКТОВАЯ ЛИНЕЙКА

### Основные услуги
- USDC платежи с автоматической конвертацией в USD

- Подписки и recurring billing
- Унифицированная отчетность
