# ДОСЬЕ: BTCPay Server

**Статус:** Open-source лидер  
**Год основания:** 2017  
**Штаб-квартира:** Децентрализованный (open-source проект)  
**Тип:** Self-hosted, некастодиальное Bitcoin-решение

## ОСНОВНАЯ ИНФОРМАЦИЯ

### Позиционирование
- Полностью бесплатный, self-hosted, open-source Bitcoin платежный процессор
- Максимальная суверенность и контроль над платежной инфраструктурой
- Нулевые посредники и полная приватность

### Поддерживаемые активы
- **Основной фокус:** Bitcoin
- **Lightning Network:** Полная поддержка
- **Расширения:** Дополнительные криптовалюты через плагины
- **Стратегия:** Bitcoin-first с возможностью расширения

## ПРОДУКТОВАЯ ЛИНЕЙКА

### Основные услуги
- Self-hosted платежный процессор
- Lightning Network интеграция
- Point-of-Sale приложения
- Invoicing система
- Crowdfunding платформа
- Payroll система

### Технические возможности
- **Полный контроль:** Собственные Bitcoin ноды
- **Приватность:** Нет KYC, нет отслеживания
- **Кастомизация:** Полная настройка под потребности
- **Интеграции:** API для custom разработки

## ЦЕНОВАЯ МОДЕЛЬ

### Структура комиссий
- **Платформенная комиссия:** $0 (полностью бесплатно)
- **Транзакционные комиссии:** $0 (только сетевые комиссии Bitcoin)
- **Hosting costs:** $10-50/месяц (cloud) или self-hosting
- **Модель:** Полностью бесплатное ПО

### Дополнительные расходы
- **Cloud hosting:** $10-50/месяц в зависимости от провайдера
- **Self-hosting:** Стоимость собственного сервера
- **Техническое обслуживание:** Время технических специалистов

## СООТВЕТСТВИЕ И БЕЗОПАСНОСТЬ

### Лицензирование
- **KYC/AML:** Нет (self-hosted решение)
- **Регулирование:** Не применимо (open-source ПО)
- **Время верификации:** Нет верификации
- **Соответствие:** Зависит от юрисдикции пользователя

### Безопасность
- **Open-source:** Полная прозрачность кода
- **Self-custody:** Полный контроль над приватными ключами
- **Децентрализация:** Нет единой точки отказа
- **Community audit:** Постоянный аудит сообществом
